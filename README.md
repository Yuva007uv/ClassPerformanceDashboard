# Student Data Analysis & Class Performance Dashboard

A comprehensive web-based analytical dashboard designed to monitor and visualize student performance, attendance, and placement readiness for the 23CS batch.

## 🚀 Overview
This project provides a data-driven approach to student academic tracking. It enables educators and administrators to gain insights into:
- **Class Academic Performance:** Average CGPA, distribution of high and low performers.
- **Attendance Tracking:** Real-time visibility into class attendance averages.
- **Demographic Insights:** Distribution of students by state, gender, and accommodation type.
- **Placement Readiness:** Eligibility counting and placement readiness scoring.

## ✨ Features
- **Interactive Visualizations:** Built with Chart.js for dynamic and responsive data representation.
- **Advanced Filtering:** Filter student data by State, Gender, Accommodation (Hosteller/Day Scholar), and Placement Eligibility.
- **Student Deep-Dive:** Detailed modal views for individual student performance metrics.
- **Premium Design:** Modern, sleek interface with glassmorphism effects and smooth transitions.
- **Responsive Layout:** Works seamlessly across desktops and mobile devices.

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3 (Vanilla CSS with modern gradients and animations)
- **Logic:** JavaScript (ES6+)
- **Charts:** [Chart.js](https://www.chartjs.org/)
- **Data Visualization:** Custom CSS gauges and performance bars.
